## Project Description
This project involves creating and managing objects representing customers, restaurants, and reviews. The goal is to understand object-oriented programming concepts and implement relationships between objects.

## Project Structure
The project consists of three main classes:

Customer: Represents a customer with attributes for their given name, family name, and a list of reviews.
Restaurant: Represents a restaurant with an attribute for its name and a list of reviews.
Review: Represents a review with attributes for the customer, restaurant, and rating.
